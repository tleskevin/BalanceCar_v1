# STM32F103C8T6 雙輪自平衡小車專案 (BalanceCar_v1)

本專案基於 **STM32F103C8T6 (Blue Pill)** 核心板開發，實現了一套具備**動態適應性、零死區捕捉與抗積分飽和（防暴衝）機制**的級聯 PID 雙輪自平衡控制系統。專案特別針對「重心偏高、重頭車」以及「加裝配件導致的物理重心不對稱」進行了演算法層面的深度優化。

## 🚀 核心控制算法與技術特點

1. **級聯 PID 控制架構 (Cascade PID Control)**
   * **內環（直立環）**：採用角度誤差分段變參數 PD 控制，小角度下引入高微分阻尼（$Kd$），提供類似空氣黏滯感的穩定度，扼殺簡諧震盪；大角度下自動加強比例剛性（$Kp$）實現急速拉回。
   * **外環（速度環）**：採用速度 PI 控制，其輸出作為直立環的「動態目標修正角度」，透過重力分量柔和引導小車定點站立或慢速平滑移動。

2. **硬體過零抑制阻尼與動態啟動補償**
   * 當車身在回正過程中跨越直立零點時，演算法會自動對舊輸出進行降額衰減（Damping），有效避免車體因慣性衝過頭而摔向另一邊。
   * 內建動態最小啟動 PWM 補償，自動跨越馬達齒輪箱的物理死區摩擦力，保證微幅調整時的敏捷響應。

3. **非線性速度抑制與抗積分飽和 (Anti-Windup)**
   * 針對非對稱配重車體，將速度環的積分天花板實施硬核限幅（鎖死在低飽和區），徹底根除小車因持續重力下墜導致速度積分無限累積、進而引發毀滅性前進暴衝的技術病灶。
   * 引入大角度非線性速度限幅，角度偏差越大，前進角度天花板收得越緊，強迫馬達在極短距離內踩住煞車回拉。

4. **精準 5ms 採樣週期對齊**
   * 放棄不穩定的軟體延時，控制主線精準錨定在由系統滴答定時器（SysTick）或硬體定時器驅動的 **5ms（200Hz）採樣週期**上，確保姿態融合互補濾波與控制算法中的微分項（$dt$）具有絕對嚴謹的時間基準。

## ⚙️ 關鍵巨集常數調校指南 (`main.c`)

專案將所有與物理車體配重、運動速度相關的參數高度抽象化為頂部的 `#define` 巨集，無需修改底層邏輯即可快速適應新配重：

* **`BALANCE_OFFSET_ANGLE` (手動平衡角偏移)**
  * *功能*：用於對齊配件變更後的「實際重力中立點」。
  * *調校*：開機放手後若緩慢往前溜，將此值朝負方向微調（如 `-1.0f`, `-1.5f`）；若往後溜，則朝正方向微調。
* **`SPEED_ANGLE_MAX` (最大移動速度限制)**
  * *功能*：鎖死速度環允許輸出的最大前傾修正角。
  * *調校*：預設設定在 `1.2f` ~ `3.0f` 的極低區間。數值越小，移動越慢越穩；數值變大，移動響應變快。
* **`PWM_MAX` (馬達硬體輸出上限)**
  * *功能*：從物理源頭限幅馬達最大輸出（範圍 0~999），防止過度前傾時馬達瞬間爆發力過大導致甩飛暴衝。
